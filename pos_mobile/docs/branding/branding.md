# Branding

## App name ideas (shortlist)

Pick one that matches your store / market:

- **SariPOS** — short, local-store friendly
- **PocketPOS** — emphasizes mobile
- **QuickSale POS** — emphasizes speed at checkout
- **TindaPOS** — “tindahan” vibe, short and memorable
- **LedgerPOS** — emphasizes tracking/records
- **KasirKit** — “cashier kit” feel
- **PayRail POS** — references the adaptive NavigationRail (subtle)
- **StockMate POS** — emphasizes inventory/stock

## Logo files

The repo includes scalable SVG logos (easy to edit, export to PNG, or use in marketing):

- `assets/branding/logo_mark.svg` (app icon / mark)
- `assets/branding/logo_lockup.svg` (mark + word-shape)

## Colors

These logos use the app’s current default brand color:

- Teal: `#005F5C` (matches the default `brand_color` in the local settings table)

## Export tips

To generate PNGs from SVG you can use any of:

- Inkscape (export at 1024×1024 and 512×512)
- Figma (import SVG, export PNG)
- Online SVG→PNG converters (for quick drafts)

For iOS/Android app icons, a common target size for source art is 1024×1024.
