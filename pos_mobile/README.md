# pos_mobile

Flutter POS mobile app.

## Quick start

Prereqs:

- Flutter (stable) with Dart `>= 3.10.4`
- Android SDK / Android Studio

Commands:

- Install deps: `flutter pub get`
- Run: `flutter run`
- Debug APK: `flutter build apk --debug`
- Release APK: `flutter build apk --release`
